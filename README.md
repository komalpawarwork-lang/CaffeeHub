# ☕ CafeHub - React Routing Project

CafeHub is a modern web application built using React.js that demonstrates dynamic client-side routing and smooth navigation between multiple pages without reloading the browser.

## 🚀 Features

- 🔀 Client-side routing using React Router
- 🧭 Smooth navigation between pages (Home, Menu, About, Contact)
- ⚡ Single Page Application (SPA) for fast performance
- 🧩 Component-based architecture for reusability
- 📱 Responsive and user-friendly UI

## 🛠️ Technologies Used

- React.js
- React Router DOM
- JavaScript (ES6+)
- HTML5
- CSS3

## 💡 Project Objective

The objective of this project is to understand and implement routing in React, enabling seamless navigation and improved user experience without full page reloads.
